残差网络论文阅读笔记(Deep Residual Learning for Image Recognition)

发表于 2018-08-23 | 分类于 CS | 阅读次数：

《Deep Residual Learning for Image Recognition》是一篇2015年发表的论文，介绍了残差网络（ResNet）

简介

  神经网络越深（十几层左右），可以提取的特征越多。但是随着网络的加深，会出现梯度弥散/梯度爆炸的问题，然而这些问题很大程度上被标准初始化（ normalized initialization）和中间归一化（ intermediate normalization layers）解决。

An obstacle to answering this question was the notorious problem of vanishing/exploding gradients [1, 9], which hamper convergence from the beginning. This problem, however, has been largely addressed by normalized initialization [23, 9, 37, 13] and intermediate normalization layers [16], which enable networks with tens of layers to start con- verging for stochastic gradient descent (SGD) with back-propagation [22].

  但开始训练更深的神经网络时，会出现一个更致命的问题：准确率会先饱和然后迅速降低。但这并不是过拟合的问题，对一个合适的模型增加网络层会导致更高的训练误差。我们便可以使用深层残差网络可以解决这个问题。

  残差网络很好优化，但是”plain net“（简单的堆叠网络层）在深度增加时会有更高的训练误差；并且残差网络随着深度的增加可以获得更高的准确率，比之前的网络架构效果更好。
